# spring-boot-cli-hello

When we installed springboot-cli, we can create a file called app.groovy to quick start a spring-boot app

[How to install springboot-cli](https://docs.spring.io/spring-boot/docs/2.1.1.RELEASE/reference/htmlsingle/#getting-started-installing-the-cli)

We can run the app by type as follows:
```bash
spring run app.groovy -- --server.port=18080
```
